About this project

UrbanFlow

An interactive city planning simulator and road network optimisation tool built for Nairobi, Kenya. Developed as a class project to explore graph algorithms, traffic modelling, and urban data visualisation.

What it does

UrbanFlow lets you load real road networks from OpenStreetMap, compute optimal routes using Dijkstra and A* algorithms, predict traffic congestion by time of day, animate vehicle simulations, and compare alternative infrastructure scenarios — all in the browser with no specialist training.

The tool is focused on Nairobi but works with any location on OpenStreetMap. Road data is fetched live, so the network reflects current real-world geometry.

Data disclaimer

Important — synthetic traffic data

The traffic congestion predictions in this tool are generated by a machine learning model trained on synthetic data — not real Nairobi traffic counts. Congestion profiles are based on typical urban patterns (morning/evening peaks, weekend reductions) and are intended for demonstration purposes only.

Road geometry and network topology come from OpenStreetMap and reflect real-world data, but may be incomplete or out of date for some areas.

API reference

The backend exposes a REST API used by the frontend:

MethodEndpointDescription
POST/api/network/fetchFetch OSM road network for a bounding box
POST/api/routing/dijkstraShortest path via Dijkstra
POST/api/routing/astarShortest path via A*
POST/api/traffic/predictCongestion prediction for hour + day
GET/api/scenarios/List saved scenarios
POST/api/scenarios/Save current network as a scenario
DELETE/api/scenarios/<id>Delete a scenario
